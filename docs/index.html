Online MIME Headers Encoder/Decoder (RFC 2047)

Output:

This tool is an online encoder/decoder for RFC 2047, the Internet standard for representing non-ASCII text in MIME message headers.

It is not intended to be used as a validator, and assumes that any supplied input conforms to the standard. In the case of non-conforming input, behaviour is undefined, although it would typically result in the encoded word (or its incorrectly-formed portions) being displayed verbatim. The use of characters sets not supported by the decoder would also result in the encoded word being displayed verbatim.
